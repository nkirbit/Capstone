About the Project

Why should I care?

My project aims to understand how changes in COVID-19 rates affect consumer's level of interest in different industries. This is worth measuring because it can inform how different businesses should respond to changing rates of COVID-19 in their location.

For example, I find that when COVID cases per 100k people increase by 100, interest in bars drops by 5.5%, interest in restaurants drops by 11%, and interest in grocery stores remains unchanged.

Methods

To measure consumer interest, I used Google Trends data. I downloaded daily data over the period from March, 2020 to the present, and peformed regression analysis to measure which search terms were influenced by changes in COVID-19. I used a distributed lag model with 4 lags for COVID-19 rates, and included categorical variables for the month, day of week, and state.

I have included results for a small sample of businesses. Users can choose a state and type of business, and make predictions describing how different paths of COVID-19 rates would influence future consumer interest.

I collected google trends data using the Pytrends package for Python. All analysis was done in Python.

Unfortunately, I am unable to let users enter their own search terms, as the process of gathering data from Google Trend's API includes several hundred requests, and would time out this page.
